Lists

Lists are used to group together related pieces of information so they are clearly associated with each other and easy to read. In modern web development, lists are workhorse elements.

Simple Ordered List

List Heading

  1. Person 1

  2. Person 2

  3. Person 3


    <div class="list">
        <h4 class="list-header">List Heading</h4>
        <ol class="simple-ordered">
            <li class="items">
                <p class="item-content">Person 1</p>
            </li>
            <li class="items">
                <p class="item-content">Person 2</p>
            </li>
            <li class="items">
                <p class="item-content">Person 3</p>
            </li>
        </ol>
    </div>
