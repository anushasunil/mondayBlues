Modal

Modals are small pop-up windows which are really handy when you want to display to the user extra content, configuration or consent request.

- Simple dialogs display a list of items

Alert

- Alert dialogs interrupt users with urgent information, details, or actions.

Modal

This website wants to access the location of your device


    <div class="modal alert display-flex-column">
        <h3 class="modal-header">Modal</h3>
        <div class="contents display-flex-column">
            <p class="contents">This website wants to access the location of your device</p>
            <div class="action-btn display-align-center display-justify-flex-end">
                <button class="solid-error bold">DENY</button>
                <button class="solid-success bold">ALLOW</button>
            </div>
        </div>
    </div>
