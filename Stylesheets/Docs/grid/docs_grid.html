Simple Grid

The property display: grid is used to position the child elements within the parent container in 2D arrangement. Grid properties like grid-templates-columns, grid-templates-rows are used to create rows and columns. With the help of grid-template-areas we can explicitly create the layout for child elements. 


<div class="grid-container">
    <div class="grid-element one"></div>
    <div class="grid-element two"></div>
    <div class="grid-element three"></div>
    <div class="grid-element four"></div>
    <div class="grid-element five"></div>
    <div class="grid-element six"></div>
    <div class="grid-element seven"></div>
    <div class="grid-element eight"></div>
</div>
