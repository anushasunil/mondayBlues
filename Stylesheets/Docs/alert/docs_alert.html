Alert

Alert are used to notify the user about any important information in a short message without interrupting user's work flow.

Primary Alert

- the standard alert box!

This is a primary alert.


    <div class="alert-box primary display-align-center">
        <div class="alert-content flex-grow">
            <p>This is a primary alert.</p>
        </div>
        <i class="fa-solid fa-xmark clickable-image"></i>
    </div>

Success Alert

- a success themed solid-type alert box can be used to give any success-notification

Task completed successfully.


    <div class="alert-box success display-align-center display-justify-space-between">
        <div class="alert-content display-align-center">
            <i class="fa-solid fa-circle-check icon"></i>
            <p class="flex-grow">Task completed successfully.</p>
        </div>
        <i class="fa-solid fa-xmark clickable-image  icon"></i>
    </div>

Warning Alert

- a solid-type warning alert box usually warns against any possible error that might occur during the execution of task

This may lead to loss of data.


    <div class="alert-box warning display-align-center display-justify-space-between">
        <div class="alert-content display-align-center">
            <i class="fa-solid fa-triangle-exclamation icon"></i>
            <p class="flex-grow">This may lead to loss of data.</p>
        </div>
        <i class="fa-solid fa-xmark clickable-image  icon"></i>
    </div>

Error Alert

- an error themed solid-type alert box indicates a fatal error

Incorrect credentials. Please enter the valid credentials.


    <div class="alert-box danger display-align-center display-justify-space-between">
        <div class="alert-content display-align-center">
            <i class="fa-solid fa-circle-exclamation icon"></i>
            <p class="flex-grow">Incorrect credentials. Please enter the valid credentials.</p>
        </div>
