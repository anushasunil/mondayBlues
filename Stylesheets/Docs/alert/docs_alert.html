Alert

Alert are used to notify the user about any important information in a short message without interrupting user's work flow.

Primary Alert

- the standard alert box!

This is a primary alert.

Success Alert

- a success themed solid-type alert box can be used to give any success-notification

Task completed successfully.

Warning Alert

- a solid-type warning alert box usually warns against any possible error that might occur during the execution of task

This may lead to loss of data.

Error Alert

- an error themed solid-type alert box indicates a fatal error

Incorrect credentials. Please enter the valid credentials.
